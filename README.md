# JazzCash Expense Tracker 💰
> Automate your JazzCash expenses. Zero apps, zero effort.

A lightweight, fully native iOS Shortcut that automatically tracks your JazzCash transactions and logs them into Apple Notes — no third party apps, no subscriptions, no manual entry.

---

## 📱 Overview

Every time you make a JazzCash payment, this shortcut automatically extracts the transaction amount, asks you to select a purpose of payment, and record a clean formatted entry into your designated Apple Notepad folder. Everything runs natively on your iPhone and stays completely within Apple's ecosystem.

---

## ✨ Features

- 🔍 Auto extracts transaction amount using Regex
- 📂 Purpose selection at runtime
- ✏️ Custom input option for purpose of payment
- 📅 Auto formatted date on every entry
- 📝 Clean log format — `Date | Amount - Category`
- 🔒 Fully private, all data stays in Apple Notes
- ⚡ Zero dependencies, no internet required

---

## 📋 Requirements

- iPhone running iOS 16 or later
- Apple Shortcuts app
- Apple Notes app
- JazzCash account

---

## 🚀 How It Works

1. JazzCash payment is made
2. Shortcut is triggered
3. Transaction amount is extracted automatically
4. User selects the purpose of transaction from a category list
5. Entry is logged into Apple Notes in this format:

```
27/02/2026 | Rs.500 - Groceries
28/02/2026 | Rs.200 - Food
```
